HTML (HyperText Markup Language) is the most basic
building block of the Web. It defines the meaning and structure of web content.
Other technologies besides HTML are generally used to describe
a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Cascading Style Sheets (CSS) is a simple mechanism for
adding style (e.g., fonts, colors, spacing) to Web documents.
These pages contain information on how to learn and use CSS and on
available software. They also contain news from the CSS working group.
